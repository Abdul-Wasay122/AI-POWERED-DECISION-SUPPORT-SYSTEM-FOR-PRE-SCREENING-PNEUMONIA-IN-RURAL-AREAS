# 🩺 AI-Powered Decision Support System for Pre-Screening Pneumonia in Rural Areas

## 📌 Overview

This project presents an **AI-powered decision support system** designed to assist in the **early pre-screening of pneumonia**, particularly in **rural and resource-limited areas**. By leveraging machine learning, the system helps identify potential pneumonia cases quickly, enabling faster medical intervention. it is compatible to run even ofline

---

## 🚀 Features

* AI-based pneumonia pre-screening
* Lightweight and easy-to-run application
* Designed for accessibility in low-resource environments
* Streamlit-based interactive interface

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install Dependencies

Make sure you have Python installed, then install the required libraries:

```bash
pip install -r requirements.txt
```

*(If `requirements.txt` is not available, install libraries manually such as `streamlit`, `numpy`, `pandas`, `tensorflow/keras`, etc.)*

---

## ▶️ How to Run the Application

1. Ensure the trained model file is in the same directory as `app.py`.

2. Run the following command in your terminal:

```bash
streamlit run app.py
```

3. The application will launch in your browser automatically.

---

## 🧠 How It Works

* The system takes medical input (e.g., chest X-ray or clinical features).
* The trained AI model processes the input.
* It outputs a prediction indicating the likelihood of pneumonia.

---

## 🌍 Use Case

This tool is particularly useful for:

* Rural healthcare centers
* Clinics with limited diagnostic resources
* Early-stage screening before expert consultation

---

## ⚠️ Disclaimer

This system is intended for **pre-screening purposes only** and should not replace professional medical diagnosis. Always consult a qualified healthcare provider for medical decisions.
